# Had pro začátečníky

Tento projekt je jednoduchá a velice primitvní hra Had (Snake). <br>
Hada ovládáš wasd, sbíráš jablka - zvětšíš se anebo hrušky - zmenšíš se, pozor ať nezemřeš. <br>
Cílem je porazit všech 10 levelů hry, aniž by had narazil do zdi nebo do sebe. <br>

Super gameplay <br>
